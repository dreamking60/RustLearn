### Compile and Execute
```sh
#compile
rustc main.rs
#execute
./main
```

### Build project by cargo
```sh
#init project
cargo new [project name]
#build project for debug
cargo build
#run the project
cargo run
#check the project's executable file
cargo check
# build for release
cargo build --release
```
